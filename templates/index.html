API Endpoints (for Postman):

  • GET /api/heroes - Get all heroes
  • GET /api/heroes/<id> - Get hero by ID
  • POST /api/heroes - Create new hero
  • PUT /api/heroes/<id> - Update hero
  • DELETE /api/heroes/<id> - Delete hero
